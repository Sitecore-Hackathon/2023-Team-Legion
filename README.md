## Description

⟹ Sitecore SXA module: RenderingConverter.

## Pre-requisites and Dependencies

-   Sitecore XM 10.3
-   .NET Core 3.1 SDK
-   .NET Framework 4.8 SDK
-   Sitecore PowerShell Extensions 6.4
-   Sitecore Headless Rendering 21.0.0
-   Sitecore Experience Accelerator 10.3.0
